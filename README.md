# Youth Financial Health Monitor

A Streamlit web application that analyzes financial health based on income and expenses for youth aged 18-24.

## Features

- 📊 Machine Learning Model for Financial Health Classification
- 💰 Real-time Financial Analysis
- 📈 Expense Breakdown Visualization
- 🎯 Personalized Financial Recommendations
- 🌐 Global Web Access

## Installation (Local)

```bash
python -m venv .venv
.\.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Mac/Linux

pip install -r requirements.txt
```

## Running Locally

```bash
streamlit run new.py
```

Then open: `http://localhost:8510`

## Deployment

This app is deployed on Streamlit Cloud at:
https://youth-financial-monitor.streamlit.app

## Models Used

- Logistic Regression
- Decision Tree
- K-Nearest Neighbors
- Gaussian Naive Bayes
- SVM (Linear & RBF)
- Random Forest
- Gradient Boosting
- XGBoost
- LightGBM
- Neural Networks (MLP)
